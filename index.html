    Beranda

    Web ini merupakan dokumentasi perkuliahan yang bersifat tidak resmi. Semua konten dalam web ini saya tulis berdasarkan pemahaman saya saat mengikuti kegiatan perkuliahan gelar sarjana di program studi Teknik Fisika, Fakultas Teknik, Universitas Gadjah Mada. Karena keseluruhan dokumentasi ini saya tulis berdasarkan pemahaman saya, maka sangat memungkinkan bagi Anda untuk menemukan kesalahan dalam berbagai konten saya. Jika Anda menemukan kesalahan dalam konten saya, maka harap kabari saya melalui surel yang saya cantumkan di bagian bawah laman ini. Saya akan berusaha untuk memperbaikinya. Terima kasih, selamat membaca :)
