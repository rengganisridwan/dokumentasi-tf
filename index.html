    Teknologi

    Pembuatan web ini terinspirasi dari PUEBI Daring karya Ivan Lanin. Web ini dibangun menggunakan Markdown, MkDocs, dan Github Pages. Berbeda dengan PUEBI Daring, web ini menggunakan Visual Studio Code sebagai editor utama dan Material for Mkdocs sebagai temanya.

    Material for Mkdocs memiliki fitur berupa tombol pintasan yang dapat digunakan ketika menelusuri web ini. Tombol F atau S pada keyboard dapat digunakan sebagai tombol pintasan untuk fitur pencarian, sedangkan tombol Esc dapat digunakan untuk keluar dari mode pencarian.
